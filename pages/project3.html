Dynamic Speed Controller

This project features a dynamic speed controller implemented on an Arduino platform. It controls the direction and speed of a motor based on user inputs from a switch and a potentiometer.

Technical Details:

  • Microcontroller: Arduino
  • Input Devices: Switches and potentiometer for speed adjustment
  • Output Devices: H-Bridge motor driver for bidirectional control

Code Explanation:

// Define pin connections
int switchPinRev = 7;     // Input from the switch for reverse direction
int switchPinFwd = 2;     // Input from the switch for forward direction
int fwdPin = 5;           // Output to the H-Bridge for forward motion
int revPin = 6;           // Output to the H-Bridge for reverse motion
int potentiometerIn;      // Variable to hold the potentiometer input

// Setup configuration, runs once on startup
void setup() {
pinMode(switchPinRev, INPUT_PULLUP);
pinMode(switchPinFwd, INPUT_PULLUP);
pinMode(fwdPin, OUTPUT);
pinMode(revPin, OUTPUT);
}

// Main loop, runs repeatedly
void loop() {
potentiometerIn = analogRead(A0);
int output = potentiometerIn / 4;  // Scale to match PWM duty cycle (0-255)

if (digitalRead(switchPinFwd) == LOW) {
analogWrite(fwdPin, output);
}

if (digitalRead(switchPinRev) == LOW) {
analogWrite(revPin, output);
}
}
